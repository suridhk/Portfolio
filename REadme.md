#### Beckett Pizza Plaza
A commercial logic script designed to handle a **"4-for-3" promotion**.
* **Key Features:** Validates numeric input, identifies the cheapest item via `min()`, and calculates the final discounted total.
* **Concepts:** Lists, input validation, and mathematical operations.

####  Password Security Check
A security simulation mimicking **partial-password authentication** used in banking systems.
* **Key Features:** Checks password length requirements and requests specific characters based on random indexing.
* **Concepts:** String manipulation, `random` library, and program termination logic.
